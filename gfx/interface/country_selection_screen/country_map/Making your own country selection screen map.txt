Hello there!

If you're reading this, then you're most likely looking to impliment your own country selection screen map for EaW, or perhaps seeking inspiration for another mod. Either way, welcome!

https://drive.google.com/drive/folders/1rakuJBqmSFsY1ANGFlxR-HXan6Fqz5gi?usp=drive_link
The three .psd files in this folder are what we used to create the maps you see in the country selection screen.
Full credit goes to Deer (lunar_explorator) for making the map. (If you're using these maps, please do make sure to credit him!)

Once you open the photoshop file, pay attention to the following
1. Creating the glow/highlight effect revolves around the "Mask" folder and the "Name mask" layer in the layers list.
2. You're going to need to mask out the "map darken" and "map desaturate" layers as we have done in the .psd files to ensure that only your country is bright in the selection.
3. We also add an extra layer on top of these two effect layers by copying the border color of the country ("Border Colors" layer) and giving it a gaussian blur to give a faint glow to the country.
4. Make sure to also mask out the "Name mask" layer to ensure that only your country's text is bright.
5. Export the completed map and place it under the map template located in this folder.

Implementation of the map in the country selection screen is by including it as part of a country's brief history localization, as a texticon.
Define this exported file as a texticon in a .gfx file.
Afterwards, implement it as part of the brief history localization like so:

<TAG>_GATHERING_STORM_DESC: "\n\n\n\n\n\n\n\n£GFX_texticon_CSSc_<TAG>_map\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n\n<...brief history description here...>"

The texticon file for base EaW is located in `interface\eaw_frontendgamesetupview.gfx`,
The localisation file for base EaW is located in `localisation\english\eaw_bookmarks_l_english.yml`

And you're all set! Have fun!
