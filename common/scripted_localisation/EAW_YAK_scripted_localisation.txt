# Allows you to create your own dynamic keys
# to be called in localization.
# defined_text -> this is it, we're defining the text
# text -> a discrete entry that can be picked to display in loc.
# trigger -> determines if a text entry will be picked or not.
# (The triggers need to be valid for the scope the key is called in
# (eg Root or From.From).)
# localization_key -> points to the localization key
# that'll be used if trigger passes

#This is a check which matches the alert id to the localisation it should display. Use trigger logic from the given YAK_alert_id (value in YAK_alert_clan_array)
#You also have YAK_alert_index which is the current index of YAK_alert_clan_array. Use this if you need to use temporary variable logic in the trigger for whatever reason.
defined_text = {
	name = YAK_alert_icon_clan_name 
	text = {
		trigger = { check_variable = { YAK_alert_clan_array^YAK_alert_index = 1 } }
		localization_key = YAK_alert_icon_clan_juyilkhai
	}
	text = {
		trigger = { check_variable = { YAK_alert_clan_array^YAK_alert_index = 2 } }
		localization_key = YAK_alert_icon_clan_noyamanan
